# Contributing to Prosthetic Outreach Initiative

## Volunteer Task Categories

### 1. Event Support (Tier 1)
- Logistics, setup, registration at outreach events
- No prior experience required
- Great for first-time volunteers

### 2. Clinical Support (Tier 2)
- Patient intake and data collection
- Survey administration
- Basic health screening assistance
- Contact: clinical@prosthetic-outreach.org

### 3. Engineering & Technical (Tier 3)
- CAD design and 3D printing of prosthetic components
- Socket fabrication assistance
- Requires technical training (see our workshop modules)
- Contact: engineering@prosthetic-outreach.org

### 4. Coordination (Tier 4)
- Task assignment and scheduling
- Inter-team communication
- Progress tracking and reporting
- Contact: coordination@prosthetic-outreach.org

### 5. Leadership (Tier 5)
- Project management and strategic planning
- Stakeholder relations and grant writing
- Contact: leadership@prosthetic-outreach.org

## Getting Started
1. Fork this repository
2. Create a branch for your task area
3. Open an issue to claim your task
4. Submit a pull request when done
